# 04. Fundamental Analysis

Subagents for financial statement analysis, valuation modeling, and macroeconomic assessment.

## Available Subagents

| Agent | Model | Description |
|-------|-------|-------------|
| [earnings-analyst](earnings-analyst.md) | sonnet | Earnings estimates, EPS surprise analysis, guidance interpretation |
| [financial-statement-analyst](financial-statement-analyst.md) | opus | Balance sheet, income statement, cash flow, and ratio analysis |
| [macro-economist](macro-economist.md) | opus | Interest rates, inflation, GDP, central bank policy analysis |
| [sector-analyst](sector-analyst.md) | sonnet | Sector rotation, competitive dynamics, and industry analysis |
| [valuation-expert](valuation-expert.md) | opus | DCF, P/E, P/B, EV/EBITDA, and comparative valuation models |

## Quick Selection Guide

| Need | Use Agent |
|------|-----------|
| Company financials deep dive | `financial-statement-analyst` |
| Fair value calculation | `valuation-expert` |
| Macro environment assessment | `macro-economist` |
| Industry/sector analysis | `sector-analyst` |
| Earnings season analysis | `earnings-analyst` |
