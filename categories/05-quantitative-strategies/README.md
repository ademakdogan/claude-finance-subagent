# 05. Quantitative Strategies

Subagents for backtesting, algorithmic trading, and quantitative strategy development.

## Available Subagents

| Agent | Model | Description |
|-------|-------|-------------|
| [algorithmic-trading-expert](algorithmic-trading-expert.md) | opus | Algo trading engine design, order management, and execution optimization |
| [backtesting-engineer](backtesting-engineer.md) | sonnet | Backtest framework design, walk-forward analysis, and simulation |
| [mean-reversion-specialist](mean-reversion-specialist.md) | sonnet | Mean reversion strategies, half-life calculation, and z-score trading |
| [momentum-strategist](momentum-strategist.md) | sonnet | Momentum factor, trend following, and breakout strategy design |
| [pairs-trading-expert](pairs-trading-expert.md) | opus | Pairs selection, cointegration testing, hedge ratio, entry/exit signals |
| [statistical-arbitrage-pro](statistical-arbitrage-pro.md) | opus | Statistical arbitrage, cointegration, and spread trading strategies |

## Quick Selection Guide

| Need | Use Agent |
|------|-----------|
| Strategy backtest & validation | `backtesting-engineer` |
| Algo trading system design | `algorithmic-trading-expert` |
| Statistical arbitrage | `statistical-arbitrage-pro` |
| Mean reversion strategies | `mean-reversion-specialist` |
| Trend following / momentum | `momentum-strategist` |
